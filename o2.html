التَّصْرِيْفُ الْاِصْطِلَاحِىُّ

"الثُّلَاثِيُّ الْمُجَرَّدُ إِلَى وَزْنِ "أَفْعَلَ

يُنْقَلُ الثُّلَاثِيُّ الْمُجَرَّدُ إِلَى وَزْنِ " أَفْعَلَ " بِزَيَادَةِ هَمْزَةِ الْقَطَعِ فِى اَوَّلِهِ؛ ١ - لِلتَّعْدِيِةِ نَحْوُ : أكْرَمْتُ زَيْدًا؛ ٢ - وَلِلدُّخُوْلِ فِى الشَّىْئِ نَحْوُ : أَمْسَى الْمُسَافِرُ، اَىْ دَخَلَ فِى الْمَسَاءِ؛ ٣ - وَلِقَصْدِ الْمَكَانِ، نَحْوُ : اَحِجَزَ زَيْدٌ، وَاَعْرَقَ عَمْرٌو؛ اَىْ قَصَدَ الْحِجَازَ وَ الْعِرَاقَ؛ ٤ - وَلِوُجُودِ مَا اشْتُقَّ مِنْهُ الْفِعْلُ فِى الْفَاعِلِ، نَحْوُ : اَثْمَرَا الطَّلْحُ؛ اَىْ وُجِدَ فِيْهِ الثَّمَرُ، وَاَوْرَقَ الشَّجَرُ، اَىْ وَجِدَ فِيْهِ وَرَقٌ؛ ٥ - وَلِلْمُبَالَغَةِ، نَحْوُ : اَشْغَلْتُ عَمْرًا، اَىْ بَالَغْتُهُ فِى شُغْلِهِ؛ ٦ - وَلِوِجْدَانِ الشَّيْئِ فِى صَفَةٍ، نَحْوُ : اَعْظَمْتُهُو اَىْ وَجَدْتُهُ عَظِيْمًا، وَاَحْمَدْتُهُ، اَىْ وَجَدْتُهُ مَحْمُودًا؛ ٧ - وَلِلصِّيْرُورَةِ، نَحْوُ : اَقْفَرَ الْبَلَدُ، اَىْ صَارَ قَفْرًا؛ ٨ - وَلِلتَّعْرِيْضِ، نَحْوُ : اَبَاعَ الثَّوْبِ، اَىْ عَرَّضَهُ لِلْبَيْعِ؛ ٩ - وَلِلسَّلْبِ، نَحْوُ : اَشَفَى الْمَرِيْضُ اَىْ زَالَ شِفَاؤُهُ؛ ١٠ - وَلِلْحَيْنُونَةَ، نَحْوُ : اَحْصَدَ الزَّرْعُ، اَىْ حَانَ حَصَادُهُ.
